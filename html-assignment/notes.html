There are basic tags that we need to know initially.

Anchor tag
Anchor tag is used when we want to add a link in our webpage. It is always in pair. "href" attribute is use to provide address along with anchor tag. We have to give relative path to connect to resources.It writes as "a" in angle brackets.
Image tag
Image tag is self closing tag. This tag is used for import the image by giving path.This tag needs two information which called attribute: (1)Path of the image we want to link with "src" attribute. (2)Description of image with "alt" attribute. It writes as "img" in angle brackets.
Form tag
Form tag is used for adding the information. It is communication between client and server.With the form we send and recieve and store data from anywhere. It have two attributes.(1)action (2)method. It writes as "form" in angle brackets.
Table tag
Table tag is used to manage data in a tabular form. Its lauout comes as a table or chart. Many organization manage their data with this method.Table itself is table container and it includes many subtags in it like caption, thread, tbody, tr, th, td. It writes as "table" in angle brackets.
List tag
If we have information that we need to show as a list, then here is the list tag. There are three types of list. (1)Unordered list (2)Ordered list (3)Description list. It writes as "list" in angle brackets.
